
## ğŸ¨ Design System

### Colors

- **Primary**: `#FF5E5E` (Vibrant Red/Pink)
- **Secondary**: `#FFDE00` (Bright Yellow)
- **Accent**: `#5E9BFF` (Azure Blue)

- **Headings**: Syne (800 weight)

- **Border Width**: 4px solid black
- **Shadow Offset**: 6px hard shadows (no blur)
